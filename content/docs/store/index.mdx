## What is Lavaz Store?

**Lavaz Store** is a lightweight yet powerful state management engine for the React ecosystem. It is designed to be `compact`, `scalable`, and `maintainable`, providing a seamless developer experience with minimal boilerplate.

By focusing on a modular architecture, Lavaz Store allows you to manage complex application states with concise code and a clean, predictable structure. It ensures absolute **type safety** across your entire data flow, significantly reducing runtime errors.

## Key Features
* **Compact Structure**: No boilerplate, just focus on your logic.
* **Type Safety**: Built with TypeScript from the ground up.
* **Scalable**: Perfect for both small apps and large enterprise systems.

---

## Architecture Overview

Lavaz Store operates on three fundamental concepts:

1.  **Boxes**: Independent modules that encapsulate state, logic, and persistence.
2.  **Stores**: A centralized registry that orchestrates multiple boxes without conflicts.
3.  **Hooks**: Optimized React integration using `useSyncExternalStore` for synchronized and tear-free rendering.
